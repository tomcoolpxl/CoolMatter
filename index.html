Hydrogen Stationary States

Make the orbital explicit before you make it beautiful.

CoolMatter is a browser-based exploration project for hydrogen stationary states. Its visual output is tied to a defined quantum-mechanical model instead of decorative particle effects.

The first implementation stays deliberately narrow: 1s and 2s, a fixed nucleus at the origin, Monte Carlo samples drawn from |psi|^2, and validation that runs independently of the viewer.

See the Model Use on GitHub Pages
Authoritative quantity: spatial probability density rho(x, y, z) = |psi_nlm|^2
Display method: sampled detection outcomes derived from the same density
Internal length unit: Bohr radius a0

Scientific posture

What the code models

Version 1 uses the standard nonrelativistic Schrödinger model for bound hydrogen states with a fixed nucleus, no external fields, and stationary eigenstates only.

What the viewer shows

The viewer renders Monte Carlo point samples drawn from |psi|^2. It does not claim to show a literal glowing cloud of matter.

Why validation matters

Normalization, node structure, deterministic sampling, and radial histogram checks exist so the viewer stays downstream of the physics instead of replacing it.

Version 1 scope

Included

1s and 2s states, a black scene, orbit controls, a nucleus marker, and sampled electron positions derived from explicit formulas.

Explicitly excluded

No proton internals, no arbitrary fog volumes, no unsupported time dependence, and no hand-authored orbital geometry.

Later extensions

2p states, scalar fields, isosurfaces, improved diagnostics, and more advanced rendering modes can be layered on the same model.

CoolMatter is a small experiment in making the meaning of a visualized hydrogen orbital clear enough that the implementation can be inspected, challenged, and extended without losing the physics.
